## 📖 Про проект (About)

**CL Launcher** — це сучасний Open Source лаунчер, розроблений з метою розвитку та популяризації українського софту.

Наша принципова мета — **витіснити російські аналоги**, якими досі користуються за інерцією, та надати гравцям безпечний інструмент, яким можна пишатися.
<br>
<h3><b>Свій до свого по своє! 🇺🇦</b></h3>

### ✨ Основні можливості
* 🔒 **Безпечна авторизація:** Підтримка входу через Microsoft Account (Xbox Live) та LittleSkin.
* ⚡ **Оптимізація:** Легкий та швидкий, споживає мінімум ресурсів системи.
* 📦 **Менеджер модів:** Інтеграція з **CurseForge** та **Modrinth**.
* 🛠 **Завантажувачі:** Підтримка Forge, Fabric, Quilt, NeoForge, OptiFine та LiteLoader.
* 🎨 **Кастомізація:** Сучасний інтерфейс (WPF UI), теми та налаштування прозорості (Mica/Acrylic).

## 🛠 Технічний стек (Tech Stack)

Проект написаний на **C#** з використанням сучасних технологій. Ми запрошуємо розробників до співпраці!

* **Framework:** .NET 8.0
* **UI Framework:** WPF (Windows Presentation Foundation)
* **UI Library:** [WPF UI](https://github.com/lepoco/wpfui) & Fluent System Icons
* **API Integrations:** Mojang Auth, CurseForge API, Modrinth API

## 🚀 Як запустити проект (Build Instructions)
*Примітка для розробників: Ми знаємо, що код ще не ідеальний. Саме тому ми стали Open Source! Якщо ви бачите, що можна покращити — не соромтеся виправляти. Ми вчимося і ростемо разом.*
Якщо ви розробник і хочете скомпілювати лаунчер самостійно:

1. **Вимоги:**
   * Visual Studio 2022 або JetBrains Rider.
   * .NET 8.0 SDK.

2. **Клонування репозиторію:**
   ```bash
   git clone https://github.com/WER-CORE/CL-OpenSource.git

3. **Збірка:**
* Відкрийте файл рішення `.sln`.
* Дочекайтеся відновлення пакетів NuGet (Restore Packages).
* Натисніть **Build Solution** (або `Ctrl + Shift + B`).
